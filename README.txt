This repository contains various codes implemententing the finite-difference time-domain (FDTD) method to compute the propagation of electromagnetic radiation from Maxwell's equations.

Suggested references:

      * Dennis M. Sullivan, "Electromagnetic Simulation Using the FDTD Method", Wiley-IEEE Press (2000) ISBN-10: 0780347471 ISBN-13: 978-0780347472

      * Allen Taflove, "Computational electrodynamics: the finite-difference time-domain method", Artech House (1995) ISBN 1-58053-832-0
